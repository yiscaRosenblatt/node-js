שיעורי בית 

1. צרו ROUTER חדש בשם GAMES
שכאשר נכנסיים לראוט בית שלו 
localhost:3001/games

מקבלים את הרשימה המלאה שמופיעה ב API הבא:

http://fs1.co.il/bus/xbox1.php

2. צרו ב ROUTE את הראוטר בשם  SINGLE שיקבל בתור PARAMS את האינדקס של המשחקים . ושימוש בו יציג את המשחק לפי התא שלו ב JSON שקיבלנו

localhost:3001/movies/single/0

למשל הכתובת הראשונה תחזיר את המשחק הראשון (בתא 0)


3. צרו ב ROUTE ראוטר בשם GENERE שיקבל בכתובת כ QUERY STRING את המשתנה שנקרא GENERE ככה שנגיע לכתובת הבאה:
localhost:3001/games/genere?genere=fighting

נקבל בחזרה את כל המשחקים שה GENERE שלהם שווה ל FIGHTING
