Docs of API

Method Path Description Parameters / Request Body
GET /toys Get all toys (paginated, 10 per page)
skip: skip number to skip
https://node-js-bicx.onrender.com/toys?skip=0
https://node-js-bicx.onrender.com/toys?skip=10
GET /toys/search Search by name or info
s: the search query to match against toy names or info.
https://node-js-bicx.onrender.com/toys/search?s=Bicycle2025
https://node-js-bicx.onrender.com/toys/search?s=red car
https://node-js-bicx.onrender.com/toys/search?s=Ramikov
GET /toys/category/:catname Get toys by category
catName: the name of the category to retrieve toys for.
https://node-js-bicx.onrender.com/toys/category/Bicycles
https://node-js-bicx.onrender.com/toys/category/Remote cars
https://node-js-bicx.onrender.com/toys/category/board games
POST /toys Add a new toy (Requires Token)

Request Body: Toy data to be added in the following format.
price: 1-999999
img_url is allowed and not required
{
name: String,
info: String,
category: String,
price: Number,
img_url: String
}

PUT /toys/:id Edit a toy (Requires Token, UserID match) idEdit: the ID of the toy to update.
Request Body: Toy data to be updated in the following format.
price: 1-999999
img_url is allowed and not required
{
name: String,
info: String,
category: String,
price: Number,
img_url: String
}
DELETE /toys/:id Delete a toy (Requires Token, UserID match)

idDel: the ID of the toy to delete.
GET /toys/prices Get toys by price range
min : minimum price to filter by .
max: maximum price to filter by.
https://node-js-bicx.onrender.com/toys/prices?min=100&max=200
GET /toys/single/:id Get a single toy by ID
id: the ID of the toy to retrieve.
https://node-js-bicx.onrender.com/toys/single/679b6fa58fbd7a48c0390571
GET /toys/count Get total number of toys https://node-js-bicx.onrender.com/toys/count
